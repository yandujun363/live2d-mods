# live2d-mods
一些mod

## 为什么作这个
为了好玩

## 安装
把仓库git clone 下来后 把 `PRO_5.0.00~5.0.005-0.1.3.jar`   
复制到 `live2d安装目录\mods`(放在`mods`文件夹中是为了方便管理)  
然后把  
`CubismEditor5.bat`  
`CubismEditor5_d3d.bat`  
`CubismViewer5.bat`  
`CubismViewer5_d3d.bat`  
这几个bat脚本中的  
`app\lib\rlm1501.jar`  
替换成  
`mods\PRO_5.0.00~5.0.05-0.1.3.jar`  
保存后就可以启动`live2d`了
